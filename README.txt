# Rajasthan Travel Guide Website

This is a production-ready single-page travel web application template designed for tourists traveling to Rajasthan. 

## Features Included:
- **Hero Banner:** Immersive introduction block showcasing Rajasthan's signature tagline "Padharo Mhare Desh".
- **History & Cultural section:** Insightful contextual layout mapping out palaces, Rajput history, and ancient temples.
- **Dynamic 3-Column Destination Grid:** Clean, responsive interactive layout breaking down the color profiles and vibes of Jaipur, Jodhpur, and Udaipur.
- **Gastronomy Segment:** Focuses on desert foods like Dal Baati Churma and royal meats like Laal Maas.
- **Interactive Budget Calculator:** Javascript engine allowing prospective travelers to pick their tier (Backpacker, Mid-range, Luxury) to generate breakdowns instantly.

## Tech Stack Used:
- Modern HTML5 Structure
- Tailwind CSS Engine (v4 CDN)
- Embedded JavaScript calculation state logic
- Google Typography (Cinzel & Outfit pairings)

## How to use:
Simply extract the contents of the zip file and double-click `index.html` to open it locally inside any web browser.
